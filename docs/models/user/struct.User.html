Expand description

表示一个用户实体。

User 结构体对应数据库中的 users 表，包含用户的基本信息和认证状态。

Fields§

§id: i32

用户的唯一标识符。

§biz_id: String

业务相关的唯一标识符。

§name: String

用户的姓名。

§mobile: String

用户的手机号码。

§email: String

用户的电子邮件地址。

§mobile_verified: bool

手机号码是否已验证。

§email_verified: bool

电子邮件是否已验证。

§created_at: NaiveDateTime

用户创建时间。

§updated_at: NaiveDateTime

用户信息最近更新时间。

§deleted_flag: bool

是否已删除用户标志。

§password: String

用户的密码（已哈希处理，确保安全存储）。

注意: 哈希后的密码不应明文存储，避免安全风险。
